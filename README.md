# MakeON Studio

A visual circuit-building web app for designing and simulating MakeON modular electronic circuits. Drag-and-drop components onto a blueprint canvas, connect pads with conductive tape, and watch your circuit come alive.

## What It Does

MakeON Studio lets you prototype circuits using the MakeON parts system — snap-together electronic modules connected by conductive tape. Place power sources, animators (lights, buzzers, motors), and controls (buttons, sensors, switches) on a canvas, then wire them up by tapping pads.

### Parts Library

**Power**
- Powerpad — coin-cell battery holder (front/back)
- Power Box Adaptor — external battery pack with dual ± outputs

**Animators**
- Glow Light White / Glow Light Color — LED modules (color variant supports 8 colors)
- Buzzer — piezo buzzer
- Motor — DC motor
- Sound Module — 4-channel audio playback board

**Controls**
- Active NIGHT / Active DAY — light-dependent sensors
- Button — momentary push switch
- Tilt Vertical / Tilt Horizontal — tilt-activated switches
- Click ON / Click OFF — lever switches (NO/NC)
- Magnet Sensor — reed switch
- ON/OFF Switch — latching toggle

### Features

- Drag-and-drop parts placement with inventory tracking
- Pad-to-pad wiring with conductive tape simulation
- Live circuit simulation with power flow visualization
- Screenshot capture (PNG) and screen recording (WebM)
- Save/load projects as JSON
- Responsive layout for desktop and tablet
- Single self-contained HTML file — no dependencies, no build step to run

## Running It

Open `index.html` (or `MakeON-Studio.html`) in any modern browser. That's it — everything is embedded in one file.

## Building from Source

The production HTML is generated by `build.js`, which inlines all icons (base64), CSS, and JavaScript into a single file and deploys to multiple targets.

```
node build.js
```

### Project Structure

```
build.js                 # Build script — assembles the single-file app
icons_v2/               # Source PNG icons (fronts + backs, 29 files)
icons_v2.txt            # Base64-encoded icons bundle
MakeON-Studio.html      # Built output (self-contained)
MakeON-Pad-Editor.html  # Interactive pad position editor tool
```
